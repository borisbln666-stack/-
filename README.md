柏龄鸟智慧居家养老管理系统 README

一、项目简介

柏龄鸟智慧居家养老管理系统，是由四川柏龄鸟养老服务有限公司自主研发的一套面向居家养老、社区养老及政府监管的综合信息化平台。

系统基于“智慧居家养老服务 + 健康管理 + 社区运营”的模式，整合物联网、人工智能、大数据分析等技术，实现对老年人安全、健康、生活、服务的全方位管理。

系统主要服务对象包括：
	•	独居老人
	•	空巢老人
	•	失能/半失能老人
	•	社区养老服务机构
	•	政府民政部门


二、系统架构

本系统采用多端协同架构，支持不同角色使用：

1. 管理端（PC后台）

用于运营管理与数据监管
功能包括：
	•	老人档案管理
	•	服务订单管理
	•	护理员管理
	•	数据统计分析
	•	预警中心
	•	补贴与结算管理



2. 护理员端（APP / 小程序）

用于一线服务人员使用
功能包括：
	•	服务接单
	•	上门签到（GPS定位）
	•	服务记录填写
	•	健康数据录入
	•	老人状态反馈



3. 家属端（小程序）

用于老人家属远程查看
功能包括：
	•	实时查看老人状态
	•	健康数据查询
	•	服务记录查看
	•	紧急提醒接收
	•	视频/语音关怀



4. 老人端（智能设备 + 语音交互）

用于老人使用
功能包括：
	•	一键呼叫
	•	语音陪伴
	•	健康监测
	•	安全预警
	•	日常提醒（吃药/吃饭）
